                                            PROJECT RUBRICA

                                            DESCRIPTION:
                                            
Il progetto  Rubrica ha come  obbiettivo di  creare un'applicazione rubrica che possa creare,salvare, 
eliminare,  cercare e modificare  un contatto.
i file interessati per l'applicazione sono MainApp.java, Rubrica.java e Contatto.java.


                                            APPLICATION  LAUNCH:

Scaricare il file zip Rubrica aprire la cartella eseguire il file MainApp.

                                            COLLABORATORS:

Usufruito della classe DateTime creata da Patric Pintescul.
Collaboratore Xu Anqi.
